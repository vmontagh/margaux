	--MooAlgorithm[=value]
		Chooses the multiobjective optimization algorithm to use for solving problem. By default, the problem is solved using IGIA.
		Possible values include:
		IGIA : Solves the problem using the Incremental Guided Improvement Algorithm
		PGIA : Solves the problem using the Partitioned Guided Improvement Algorithm
